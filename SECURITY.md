# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in FoundryHQ, please report it
privately by emailing **odapo138@gmail.com**. Do not open a public GitHub
issue for security reports.

Please include as much of the following as you can:

- A description of the vulnerability and its potential impact
- Steps to reproduce, or a proof of concept
- The affected component (e.g. `apps/api`, `apps/web`) and version/commit

We'll acknowledge your report and follow up as we investigate. Since this
project is in early development, we can't commit to a fixed response-time
SLA yet, but reports are taken seriously and prioritized.

## Supported Versions

FoundryHQ is pre-release and does not yet have tagged versions. Security
fixes are applied to `main`.
